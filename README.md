# COPOMA — Portal informativo (Demo)

PWA informativa de la **Comunidad Portuaria de Manzanillo A.C. (COPOMA)**.

- **100% informativa.** Sin registro de usuarios, sin admin, sin transacciones.
- Datos hardcodeados de ejemplo con fines de demostración.

## Pantallas
1. **Inicio** — hero institucional, comunicado destacado y cifras del puerto.
2. **Estatus** — tablero de saturación/disponibilidad por terminal (informativo, carga manual).
3. **Comunidad** — Comité Ejecutivo + 4 Sub-Comités.
4. **Avisos** — tablón de comunicados.
5. **Contacto** — dirección, teléfono, correo, redes.

## Stack
Next.js 14 (App Router) · TypeScript · Framer Motion · PWA (manifest + sw.js) · SVG inline propios.

## Dev
```bash
npm install
npm run icons   # genera íconos PWA desde el logo
npm run dev
```
